## Installation

``` bash
pip3 install fancify-text
```

## Usage

### In Python

``` python3
>>> from fancify_text import bold, reversed_, heavyCircled, smallCaps  # Or any of the listed names above, or * to get all of them
>>> print(bold("hello world"))
```
